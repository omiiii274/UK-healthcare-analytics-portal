# Health Economic Model Dashboard

This is an R Shiny app designed for cost-effectiveness analysis. The app features various tools and visualizations to assist researchers and analysts in understanding economic models in health care.

## Key Features:

- **Tornado Diagrams**: Visual representations that illustrate the sensitivity of the results to changes in input parameters.
- **Cost-Effectiveness Plans (CE planes)**: Graphical representations that help in analyzing the trade-offs between costs and health effects.
- **Cost-Effectiveness Acceptability Curves (CEACs)**: Tools to evaluate the probability that a given intervention is cost-effective at various willingness-to-pay thresholds.
- **PostgreSQL Backend**: Efficient storage solution for model run data, allowing for scalable and reliable access to historical analyses.

This app aims to provide researchers with intuitive tools to support decision-making in health economics.
